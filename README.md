# Marvel Nexus

A minimal dark web platform for exploring MCU movies, characters, actors, discussions, leaks, facts, tags, timelines, and admin curation.

## Phase Coverage

- Phase 1: Movies, Characters, Actors, home feed, explore, entity hubs, timeline, tags, and universal search.
- Phase 2: Seeded discussions, theories, leaks, create post flow, basic profile, save toggles, and reports as moderation inputs.
- Phase 3: Admin dashboard structure for movies, characters, actors, timeline, posts, leaks, facts, tags, featured content, users, reports, global settings, and SEO metadata.

## Stack

- React
- Vite
- Lucide React
- Local seed data in `src/data.js`

## Run Locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
